## 🎯 项目简介

这是一个基于财报分析的智能投资决策辅助系统，面向投资小白和散户投资者。

**核心功能**：
- ✅ 自动获取 A股/港股/美股财报
- ✅ 支持 Tushare 和 AKShare 双数据源
- ✅ 三层架构（数据层/分析层/应用层）
- ✅ SQLite 数据存储，支持数据诊断和修复
- 🚧 基于预设分析框架自动生成分析报告（开发中）
- 🚧 集成 LLM 进行深度文本分析（规划中）

**当前进度**：Phase 0-3 已完成 ✅

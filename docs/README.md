## 🎯 项目简介

这是一个基于财报分析的智能投资决策辅助系统，面向投资小白和散户投资者。

**核心功能**：
- 自动获取 A股/港股/美股财报
- 基于预设分析框架自动生成分析报告
- 支持自定义分析框架
- 集成 LLM 进行深度文本分析
- 提供投资建议和风险提示

**当前进度**：Phase 0 - 基础建设中

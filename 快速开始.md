# 快速开始指南

## 📋 前提条件

- ✅ Python 3.9 或更高版本
- ✅ 能够上网的电脑
- ✅ 无需服务器，无需付费

## 🚀 第一步：安装依赖

```bash
cd financial_report_scraper
pip install -r requirements.txt
```

## 📝 第二步：配置文件（可选）

### A股和港股：不需要配置，直接可用！

### 美股：需要配置邮箱

1. 复制配置文件模板：
```bash
cp config.json.example config.json
```

2. 编辑 `config.json`，把邮箱改成你的真实邮箱：
```json
{
  "user_email": "your_email@gmail.com"  // 改成你的邮箱
}
```

**为什么需要邮箱？** SEC（美国证券交易委员会）要求使用者提供邮箱，这是他们的规定。

## 🎯 第三步：下载财报

### 方法1：运行演示脚本（推荐）

```bash
python demo_download.py
```

按照提示选择：
- 输入 `1` - 下载A股财报（如贵州茅台）
- 输入 `2` - 下载港股财报（如腾讯控股）
- 输入 `3` - 下载美股财报（如苹果，需要先配置邮箱）
- 输入 `4` - 全部演示

### 方法2：运行测试脚本

```bash
python test_phase0.py
```

这会测试所有功能，但不会实际下载财报。

### 方法3：在Python代码中使用

```python
from src.scrapers import scrape_a_stock, scrape_hk_stock, scrape_us_stock

# 下载A股财报
files = scrape_a_stock("600519", lookback_days=365)
print(f"下载了 {len(files)} 份财报")

# 下载港股财报
files = scrape_hk_stock("00700", lookback_days=365)

# 下载美股财报（需要先配置邮箱）
files = scrape_us_stock("AAPL", lookback_days=365)
```

## 🎓 新手提示

1. **先试试A股下载**：不需要任何配置，最简单
2. **查看下载的文件**：在 `reports/` 目录里
3. **不用担心费用**：目前所有功能都免费
4. **网络问题**：如果下载失败，可能是网络问题，多试几次

## 🐛 常见问题

### 1. 美股下载403错误
- 原因：SEC网站有访问限制
- 解决：确保配置了真实邮箱，稍后重试

### 2. 找不到config.json
- 解决：先运行 `cp config.json.example config.json`

### 3. 模块导入错误
- 解决：确保安装了依赖 `pip install -r requirements.txt`
